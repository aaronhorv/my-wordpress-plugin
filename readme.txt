A simple WordPress plugin.

== Description ==

This is a starter WordPress plugin with a basic structure that you can build upon.

Features:

* Clean plugin architecture
* Translation ready
* Asset enqueueing setup
* Activation and deactivation hooks

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/my-wordpress-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= How do I customize this plugin? =

Edit the main plugin file and add your custom functionality in the includes/functions.php file.
